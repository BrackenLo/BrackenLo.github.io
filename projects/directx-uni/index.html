CPP Graphics University

"Graphics 1" university module learning DirectX 11. There's nothing too much here outside of course content with the final assessment being a demonstration of standard Blinn-Phong lighting and a basic scene graph.

The only thing of mine to really note was the university provided a DirectX project template which setup and window and DirectX context for us. I wanted to create my own version to get an understanding of how the project was structured and what did what.

While I never got far enough in to implement shaders and actually see it work, it still helped to get an understanding DirectX. It also would have been ineligile for the assessment criteria requiring the provided code.
