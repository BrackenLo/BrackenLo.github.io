I am a recent graduate from Derby University with a 2:1 in Computer Games Programming. Although the university taught me to use the Unreal Engine and C++, I have a hands-on approach to learning and have taught myself to use other game engines and other programming languages. This portfolio includes examples written across languages such as Rust, C#, C++ and GDScript and engines such as Unreal, Godot, Unity and Bevy.

This portfolio is very new and very work in progress. My Github account contains most of the code available on this portfolio. Also see my old github account (still in the process of migrating work).
